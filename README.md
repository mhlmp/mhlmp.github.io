## 🎯 מה האתר הזה עושה?

**MH Link Manager Pro** הוא הרבה יותר מסתם רשימת קישורים. זוהי אפליקציית Web מלאה (PWA) שנועדה להחליף את המועדפים (Bookmarks) המסורתיים. היא מאפשרת לשמור כל קישור או פקודת טקסט, ולתת לבינה המלאכותית לקרוא את האתר שמאחורי הקישור, לסכם אותו למשפט אחד או שניים ברובריקה, לתייג אותו בהאשטאגים חכמים ולבדוק האם האתר זמין או "מת" (Dead/Parked). 

המערכת מסתנכרנת בזמן אמת בין המחשב לנייד דרך הענן, אבל מסוגלת לעבוד באופן מושלם גם **ללא חיבור אינטרנט (100% Offline)**.

## ✨ יכולות ויחודיות המערכת

### 🤖 בינה מלאכותית (AI) ואוטומציה
- **מנוע סריקה מרובה ספקים:** תמיכה אינטגרלית ב-**Google Gemini**, **Groq (Llama 3)**, ו-**OpenRouter**. ניתן לעבור ביניהם בלחיצה אחת.
- **סריקת עומק (Web Scraping):** יכולת לחלץ תוכן מאתרים גם אם הם מוגנים חלקית (Anti-Bot Bypass) באמצעות מערך Proxies חכם.
- **אוטומציה מלאה:** אפשרות להגדיר סריקת AI אוטומטית לכל פריט חדש שנוסף למאגר.
- **פענוח סטטוס:** זיהוי אוטומטי האם הקישור חי ופעיל (🟢), דומיין חניה (🟠) או שבור (🔴).

### ☁️ סנכרון, אופליין וניהול רשת
- **Push Queue Architecture:** מנגנון "תור" חכם מבוסס Firebase שמונע דריסת נתונים (Race Conditions) כששומרים מידע במקביל ממספר מכשירים.
- **Offline First:** כל מסד הנתונים נשמר ב-IndexedDB המקומי. האפליקציה עולה מיד ומאפשרת קריאה ועריכה גם במצב טיסה.
- **ייבוא / ייצוא (JSON & TXT):** אפשרות להוריד גיבוי מקומי מלא של הנתונים, כולל ייצוא לקובץ טקסט עם קידוד השומר על ירידות השורה (Newlines) המקוריות של סיכומי ה-AI.

### 🎨 ממשק משתמש (UI) וחוויית משתמש (UX)
- **פעולות גורפות (Bulk Actions):** מצב Multi-Select המאפשר בחירת עשרות פריטים למחיקה, העברה לקטגוריה אחרת או סריקת AI המונית.
- **עיצוב Glassmorphism:** ממשק זכוכית יוקרתי עם תמיכה מלאה וחלקה במעבר בין **מצב לילה (Dark Mode)** למצב יום.
- **התראות צפות חכמות:** מערכת Toasts המציגה חיוויים על הצלחה, שגיאות, או התקדמות סריקת AI, משולבת עם כפתור "גלול למעלה" דינמי שזז כדי לא להסתיר מידע.

### 🛡️ אבטחה ופרטיות
- **הגנת כלי מפתחים (DevTools Protection):** מנגנון ייעודי החוסם כניסה לקונסול ול-Inspect Element כדי להקשות על העתקת קוד או הוצאת נתונים למשתמשים לא מורשים.
- **סל מחזור חכם:** פריטים שנמחקים לא נעלמים מיד, אלא עוברים לאזור מגודר שמאפשר שחזור או מחיקה לצמיתות.

## 📁 מבנה הקבצים

| קובץ | תפקיד |
| :--- | :--- |
| `index.html` | האפליקציה המלאה (HTML + CSS + JS Vanilla), ממשק המשתמש והלוגיקה |
| `manifest.json` | הגדרות PWA המאפשרות התקנה כאפליקציה טבעית (Native) על שולחן העבודה והנייד |
| `sw.js` | Service Worker האחראי על אסטרטגיית המטמון (Network First) |
| `icon.png` | סמל המערכת ברזולוציה גבוהה |
